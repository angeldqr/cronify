# Documentación de Cronify Backend

Esta carpeta contiene la documentación técnica del proyecto.

## Archivos

### API_ENDPOINTS.md
Documentación completa de todos los endpoints de la API REST:
- Ejemplos de peticiones y respuestas
- Parámetros de búsqueda y filtrado
- Códigos de estado HTTP
- Formato de datos

### CONFIGURACION_EMAIL.md
Guía paso a paso para configurar el servicio de email:
- Generar contraseñas de aplicación en Gmail
- Configurar variables de entorno
- Solución de problemas comunes

### VALIDACIONES_IMPLEMENTADAS.md
Lista de todas las validaciones del sistema:
- Validaciones de usuario (contraseñas, emails)
- Validaciones de eventos (asunto, fechas, archivos)
- Ejemplos de datos válidos e inválidos

### ESTADO_PROYECTO.md
Estado actual del desarrollo del proyecto:
- Funcionalidades completadas
- Funcionalidades pendientes
- Progreso por requisito funcional
- Checklist de implementación
